        |__ index.html

        |__ view_posts.html
    |__venv

    |__ requirements.txt
    
### Running the app
1. Clone the repository 

        git clone https://github.com/AGHARIANS/Jelly-Pot.git
        
2. Install the requirements by running:

        pip install -r requirements.txt
  
3. Run app.py

        python app.py
